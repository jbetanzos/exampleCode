# exampleCode
```
Requirement
-----------
 Having an array with N elements as input
 a. Find the elements that appear an even number of times (twice, 4 times, 6 times etc)
 b. Add an optional limit M to only return the first M elements that appear an even number of times
 Use any structure you find appropriate

Examples
--------
 [1, 3, 5, 7, 11, 2, 6, 3, 20, 50, 11, 64, 19, 99, 10000, 55] should return [3,11]
 [1, 3, 5, 7, 11, 2, 6, 3, 20, 50, 11, 1, 50] with limit 2 should return [3,11]
```
